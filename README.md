# Dotfiles

Personal Ansible roles for installing i3 on Manjaro KDE and applying my dotfiles. 

#### Usage
```
bash -c "$(wget -O- https://raw.githubusercontent.com/dvragulin/dotfiles-public/master/bin/bootstrap.sh)"
```

  - Wget
