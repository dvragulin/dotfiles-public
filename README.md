# Dotfiles

Personal ansible roles for aply i3/KDE configuration on Manjaro Linux KDE.

#### Usage
```
wget -c https://raw.githubusercontent.com/dvragulin/dotfiles-public/master/bin/bootstrap.sh && chmod +x ./bootstrap.sh && ./bootstrap.sh
```
#### Available roles

  - _global
  - alacritty
  - avatars
  - bash
  - feedbro
  - git
  - gtk
  - i3
  - mc
  - polybar
  - ranger
  - rofi
  - ssh
  - tmux
  - vim
  - wallpapers
  - xinit
  - zathura
  - zsh
