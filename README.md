# Dotfiles

Personal ansible roles for configuring my Manjaro Linux environment

#### Usage
```
wget -c https://raw.githubusercontent.com/dvragulin/dotfiles-public/master/bin/bootstrap.sh && chmod +x ./bootstrap.sh && ./bootstrap.sh
```
